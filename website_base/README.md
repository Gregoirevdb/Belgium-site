# Site Web de Base

Ce projet contient une structure de base pour un site web utilisant HTML, CSS et JavaScript. Vous pouvez l'étendre en ajoutant du contenu, des styles et des fonctionnalités.

## Structure des fichiers

- `index.html` : Page principale avec la structure HTML.
- `style.css` : Feuille de style pour l'apparence.
- `script.js` : Script JavaScript pour l'interactivité.

## Comment lancer le site

1. Ouvrez un terminal dans le dossier `website_base`.
2. Lancez un serveur local :
   ```bash
   python3 -m http.server 8001
   ```
3. Ouvrez votre navigateur et allez à `http://localhost:8001`.

## Personnalisation

- **HTML** : Modifiez `index.html` pour ajouter du contenu (textes, images, etc.).
- **CSS** : Éditez `style.css` pour changer les couleurs, polices, mises en page.
- **JavaScript** : Ajoutez des fonctionnalités dans `script.js` (formulaires, animations, etc.).

## Exemples d'ajouts

- Ajouter des images : `<img src="image.jpg" alt="Description">`
- Créer un formulaire : Utilisez `<form>` avec des `<input>` et `<button>`.
- Animer des éléments : Utilisez CSS transitions ou JavaScript.

Pour des sites plus avancés, considérez des frameworks comme React ou Vue.js.
